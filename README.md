# نظام إدارة الصيدليات

نظام لإدارة عمليات البيع والشراء والتخزين في الصيدلية. التقنيات المستخدمة في هذا النظام هي:
- Laravel 12
- Filament 4
- MySQL

## متطلبات التثبيت
- PHP ^8.2
- MySQL
- Apache or NGINIX
- Composer

لتثبيت النظام قم اولاً بعمل نسخة من هذا المستودع (Repo) على جهازك
ثم قم بتنفيذ الاوامر التالية:

```
composer install
```

```
npm install
```

- انشاء مستخدم ادمن
```
php artisan make:filament-user
```

## فكرة النظام
يقوم النظام بتخزين منتجات الادوية وترتيبها في اقسام. ويقوم بإدارة المخزون بطريقة الدفعات (Batches) وكل دفعة مخصصة لمنتج واحد فقط.

كل دفعة (Batch) تقوم بتخزين معلومات حول الكمية التي وصلت الى الصيدلية (initial_quantity) والكمية المتبقية (current_quantity) ويتم حساب الكمية باستخدام اصغر وحدة تخزين.

عند انشاء فاتورة يقوم النظام بالتأكد ان كل منتج متوفر بالكمية المطلوبة من خلال استعلام قاعدة البيانات لكل دفعات المنتج ويتم خصم الكمية من الدفعات على حسب اقدم تاريخ انتهاء (FEFO - First Expiry First Out)

يمكن ايضاً التعديل على كمية المنتج اذا وجد الصيدلي ان هناك خطأ في الارقام ويتم تسجيل ذلك في جدول adjustments

كل عملية شراء او بيع او تعديل على المخزون يتم تسجيلها في جدول stock_movements وهذا الجدول لا يمكن للمستخم التعديل عليه نهائياً.
